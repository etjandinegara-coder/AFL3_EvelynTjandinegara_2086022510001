Secara singkat tugas AFL3 ini berisi portofolio yang dibuat dengan html, javascript, dan bootstrap (CSS). Website terdiri dari 6 halaman yang saling terhubung melalui navbar.

Halaman About berisi deskripsi singkat dan experience saya sebagai programmer.
Halaman Contact berisi contact saya dan form yang bisa diinput oleh user agar saya dapat mengetahui ada yang tertarik dan bisa sy kontak. Pesan dapat sy liat melalui inspect.
Halaman Gallery berisi kumpulan-kumpulan screenshoot beberapa hasil karya.
Halaman Index berisi page home yang menampilkan tampilan awal dari website saya.
Halaman Project berisi 3 project saya yang dilengkapi dengan gambar, deskripsi singkat dan button untuk pindah ke gallery.
Halaman Skills berisi presentase beberapa skill saya.
